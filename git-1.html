## Configuración - Por repositorio - Global ao usuario (--global) ### Nome e email do usuario ``` git config --global user.name 'Nome Apelido' git config --global user.email 'nomeapelido@correo.sample' ``` ### Configuración activa ``` git config --list ``` ![GitHub](img/git_config.png)
## Área de traballo e repositorio - Área de traballo: onde fedellamos - Repositorio: baixo control --- ### Crear un repositorio ``` git init . _ ``` ### Copiar (clonar) un repositorio ``` git clone URL_REPO _ ``` --- ``` git clone https://github.com/rafacouto/git-minicurso.git ```
### Historial de cambios ``` git log _ ``` --- ### Moverse entre versións ``` git checkout _ ```
### Examinar cambios no área de traballo ``` git status _ ``` --- ### Creación dun paquete de cambios ``` git commit _ ```
### Revisión de cambios ``` git diff _ ```
